# AI Models Leaderboard Skill

🏆 OpenClaw skill for AI model rankings from Artificial Analysis.

## Overview

Fetches real-time AI model rankings from [artificialanalysis.ai](https://artificialanalysis.ai). Displays:
- Intelligence Index rankings
- Performance benchmarks
- Model comparisons

## Usage

When user mentions: `AI模型榜单`, `AI排行榜`, `大模型对比`, `最佳LLM`, `模型排名`, `leaderboard`

## Features

- Real-time rankings from Artificial Analysis
- Markdown table output
- Performance metrics (Intelligence Index, throughput, cost)
- Sent directly to current conversation

## Requirements

- OpenClaw agent
- Web access

## Installation

This skill is managed by OpenClaw. Install via:
```bash
openclaw skills install ai-models-leaderboard
```
